    Actividades Propuestas Imágenes.

    1. Diferencias entre la etiqueta title y alt.

      2. El atributo title es una etiqueta o atributo el cual es el encargado de ofrecer una descripción emergente de la imagen.

      La etiqueta alt define el texto alternativo de la imagen para en el caso de que esta no pueda ser visualizada mostrar el texto que tu le indiques.

    2. ¿Para que se utilizan los elementos map y area?

      3. Area sirve para crear secciones en las que se puede dividir un mapa de imágenes, elemento antiguo con el que se puede hacer mapas simples y poco accesibles.

      Map se utiliza para definir un mapa de imágenes, con áreas en las que se puede hacer clic, esta asociado con el atributo usemap de img creando una relaciones entre una imagen y un mapa.
